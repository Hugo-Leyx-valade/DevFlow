# Roadmap Backend Engineer — Objectif VIE Backend Europe (Suède / Backend Engineer)

## Objectif

Construire un profil crédible de Backend Engineer moderne capable de :
- décrocher un VIE backend
- réussir des entretiens techniques
- développer une application backend scalable
- comprendre les bonnes pratiques d’architecture logicielle

---

# Stack recommandée

## Stack principale

- TypeScript
- Node.js
- NestJS
- PostgreSQL
- Prisma ORM
- Docker

## Stack secondaire (bonus)

- Redis
- GitHub Actions
- Swagger
- WebSockets
- AWS / Railway / Render

---

# Compétences à développer

## 1. Backend Fundamentals

### À maîtriser
- HTTP
- REST APIs
- CRUD
- Status codes
- Headers
- Cookies
- Middleware
- Validation
- Gestion des erreurs

### Objectif
Être capable de construire une API backend propre.

## 2. TypeScript

### À maîtriser

- interfaces
- generics
- async/await

- decorators
